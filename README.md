Sample Project Structure
========================
An example project structure.

## Installation

Initialise a virtualenv:

    virtualenv .

Install with [pip](https://pip.pypa.io/en/stable/):

    pip install -r requirements.txt

Run app with:

    python3 helloworld.py

Structure:

    helloworld/
    |
    |--.gitignore
    |-- helloworld.py
    |-- LICENSE
    |-- README.md
    |-- requirements.txt
    |-- setup.py
    |-- tests.py
